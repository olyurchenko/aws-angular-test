Application Description

Todo List App is a modern web application for task management, built using Angular 18 and Angular Material. The app provides a convenient interface for creating, editing and tracking task completion progress.

Main Features

  • ✅ Adding new tasks
  • ✅ Editing existing tasks
  • ✅ Marking tasks as completed
  • ✅ Deleting tasks
  • ✅ Displaying statistics
  • ✅ Separating active and completed tasks

Architecture

The application is built using modern Angular practices:

  • Standalone Components - modular architecture
  • Angular Signals - reactive state management
  • Lazy Loading - loading optimization
  • Responsive Design - adaptive design
  • Material Design - modern UI/UX
