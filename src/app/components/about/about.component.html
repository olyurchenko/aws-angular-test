Опис додатку

Todo List App - це сучасний веб-додаток для управління завданнями, створений з використанням Angular 17 та Angular Material. Додаток надає зручний інтерфейс для створення, редагування та відстеження прогресу виконання завдань.

Основні функції

  • ✅ Додавання нових завдань
  • ✅ Редагування існуючих завдань
  • ✅ Позначення завдань як завершених
  • ✅ Видалення завдань
  • ✅ Відображення статистики
  • ✅ Розділення на активні та завершені завдання

Технології

code Angular 17
palette Angular Material
memory Angular Signals
language TypeScript
brush SCSS
bug_report Jest Testing

Архітектура

Додаток побудований з використанням сучасних практик Angular:

  • Standalone Components - модульна архітектура
  • Angular Signals - реактивне управління станом
  • Lazy Loading - оптимізація завантаження
  • Responsive Design - адаптивний дизайн
  • Material Design - сучасний UI/UX
